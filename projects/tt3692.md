## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| uo[0:7] | output | 8 |
| uio[0] | output | 1 |
| uio[1] | output | 1 |
| uio[2] | input | 1 |
| uio[3] | output | 1 |
