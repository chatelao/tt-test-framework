|--------|------|-------|
| SPI_MISO | input | 1 |
| GPIO_IN | input | 6 |
| UART0_RX | input | 1 |
| UART0_TX | output | 1 |
| ERROR_FLAG | output | 1 |
| GPIO_OUT | output | 6 |
| UIO | inout | 8 |
| CLK | clock | 1 |
| RST_N | input | 1 |
