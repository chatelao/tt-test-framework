## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| RST_N | input | 1 |
| LEFT | input | 1 |
| RIGHT | input | 1 |
| ROTATE | input | 1 |
| DOWN | input | 1 |
| R | output | 2 |
| G | output | 2 |
| B | output | 2 |
| VSYNC | output | 1 |
| HSYNC | output | 1 |
