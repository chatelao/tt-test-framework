## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |

## First 10 Cycles

| Cycle | Phase | ui_in | uo_out |
|-------|-------|-------|-------|
| 0 | Initial state | 0x0 (input a=0, input b=0) | 0x0 (output and=0, output nand=0, output or=0, output clock div 16=0, output clock div 12=0) |
| 1 | Set input | 0x1 (input a=1, input b=0) | 0x0 (output and=0, output nand=0, output or=0, output clock div 16=0, output clock div 12=0) |
| 2 | Clock pulse | 0x1 (input a=1, input b=0) | 0x0 (output and=0, output nand=0, output or=0, output clock div 16=0, output clock div 12=0) |
