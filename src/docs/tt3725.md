## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| UI_IN | input | 8 |
| UO_OUT | output | 8 |

## First 10 Cycles

| Cycle | Phase | UI_IN | UO_OUT |
|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 |
| 1 | All Inputs High | 0xff | 0x1 |
| 2 | Alternating Pattern | 0xaa | 0x0 |
