# Tiny Tapeout chip

**Source:** [https://github.com/Birkwk/Chip](https://github.com/Birkwk/Chip)

**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/3716](https://app.tinytapeout.com/projects/3716)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| RST_N | input | 1 |
| UI_IN | input | 8 |
| UO_OUT | output | 8 |

## First 10 Cycles

| Cycle | Phase | RST_N | UI_IN | UO_OUT |
|-------|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 | 0x0 |
| 1 | Load 0 | 0x1 | 0x0 | 0x0 |
| 2 | Load 1 (IN0=1, IN1=1) | 0x1 | 0x3 | 0x0 |
| 3 | Wait | 0x1 | 0x0 | 0x0 |
| 4 | Toggle | 0x1 | 0x3 | 0x0 |
