## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| UI_IN | input | 8 |
| CLK | input | 1 |
| RST_N | input | 1 |
| UO_OUT | output | 8 |

## First 10 Cycles

| Cycle | Phase | UI_IN | CLK | RST_N | UO_OUT |
|-------|-------|-------|-------|-------|-------|
| 0 | Select 0 | 0x0 | 0x0 | 0x0 | 0x1 |
| 1 | Select 1 | 0x1 | 0x0 | 0x1 | 0x2 |
| 2 | Select 2 | 0x2 | 0x0 | 0x1 | 0x4 |
| 3 | Select 3 | 0x3 | 0x0 | 0x1 | 0x8 |
