## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |

## First 10 Cycles

| Cycle | Phase | ui_in | uo_out |
|-------|-------|-------|-------|
| 0 | Initial state | 0x0 (input a=0, input b=0) | 0x0 (output 0=0) |
| 1 | Logic Test | 0xf (input a=1, input b=1) | 0x0 (output 0=0) |
| 2 | Pass-through Test | 0xf0 (input a=0, input b=0) | 0x0 (output 0=0) |
