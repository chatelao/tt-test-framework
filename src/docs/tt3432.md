## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui[0] | input | 1 |
| ui[1] | input | 1 |

| uo[7] | output | 1 |
| uo[4:6] | output | 3 |
| uo[0:2] | output | 3 |
