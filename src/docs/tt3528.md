| SO_N | input | 1 |
| GPIOA | inout | 6 |
| PHI1 | output | 1 |
| PHI2 | output | 1 |
| RW | output | 1 |
| SYNC | output | 1 |
| MUX_DATA | inout | 8 |
| CLK | clock | 1 |
| RST_N | input | 1 |
