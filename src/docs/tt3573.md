## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| UI_IN | input | 8 |
| UO_OUT | output | 8 |

## First 10 Cycles

| Cycle | Phase | UI_IN | UO_OUT |
|-------|-------|-------|-------|
| 0 | All zeros | 0x0 | 0x40 |
| 1 | Set ui0 | 0x1 | 0x1 |
| 2 | Set ui1 | 0x2 | 0x0 |
| 3 | Set ui7 | 0x80 | 0x1 |
