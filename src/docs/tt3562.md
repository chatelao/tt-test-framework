# 7-Segment Adder (0, 1, 2)

**Source:** [https://github.com/Ancash/tt](https://github.com/Ancash/tt)

**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/3562](https://app.tinytapeout.com/projects/3562)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui[0] | input | 1 |
| ui[1] | input | 1 |
| uo[0:6] | output | 7 |

## First 10 Cycles

| Cycle | Phase | ui[0] | ui[1] | uo[0:6] |
|-------|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 | 0x0 |
| 1 | 0 + 0 = 0 | 0x0 | 0x0 | 0x3f |
| 2 | 1 + 0 = 1 | 0x1 | 0x0 | 0x6 |
| 3 | 0 + 1 = 1 | 0x0 | 0x1 | 0x6 |
| 4 | 1 + 1 = 2 | 0x1 | 0x1 | 0x5b |
