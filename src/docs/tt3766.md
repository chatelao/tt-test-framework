# 8-bit SEM Floating-Point Multiplier

**Source:** [https://github.com/DelosReyesJordan/ttihp26a-FP8-SEM-Multiplier](https://github.com/DelosReyesJordan/ttihp26a-FP8-SEM-Multiplier)

**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/3766](https://app.tinytapeout.com/projects/3766)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| RST_N | input | 1 |
| ENA | input | 1 |
| A | input | 8 |
| B | inout | 8 |
| RESULT | output | 8 |

## First 10 Cycles

| Cycle | Phase | RST_N | ENA | A | B | RESULT |
|-------|-------|-------|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 | 0x0 | 0x0 | 0x0 |
| 1 | Zero Detection (0 * 1.0) | 0x1 | 0x1 | 0x0 | 0x38 | 0x0 |
| 2 | NaN Detection (NaN * 1.0) | 0x1 | 0x1 | 0x7f | 0x38 | 0x0 |
| 3 | Normal Multi (1.0 * 1.0) | 0x1 | 0x1 | 0x38 | 0x38 | 0x0 |
| 4 | Normal Multi (2.0 * 1.5) | 0x1 | 0x1 | 0x40 | 0x3c | 0x0 |
| 5 | Disabled | 0x1 | 0x0 | 0x0 | 0x0 | 0x0 |
