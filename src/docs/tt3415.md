# SimProc (Simple Processor)

**Source:** [https://github.com/ieeeuoft-asic/simproc](https://github.com/ieeeuoft-asic/simproc)

**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/3415](https://app.tinytapeout.com/projects/3415)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |
| uio_in | input | 8 |
| uio_out | output | 8 |
| clk | clock | 1 |
| rst_n | input | 1 |

## First 10 Cycles

| Cycle | Phase | ui_in | uo_out | uio_in | uio_out | rst_n |
|-------|-------|-------|-------|-------|-------|-------|
| 0 | Reset | 0x1 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x0 |
| 1 | Reset | 0x1 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x0 |
| 2 | Reset | 0x1 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x0 |
| 3 | Reset | 0x1 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x0 |
| 4 | Config BAUD | 0x1 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x1 |
| 5 | UART Start Bit | 0x0 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x1 |
| 6 | UART Start Bit | 0x0 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x1 |
| 7 | UART Start Bit | 0x0 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x1 |
| 8 | UART Start Bit | 0x0 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x1 |
| 9 | UART D0 (0) | 0x0 | 0x0 | 0x1 (BAUD[7:0]=1) | 0x0 | 0x1 |
