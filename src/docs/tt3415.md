**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/3415](https://app.tinytapeout.com/projects/3415)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |
| uio_in | input | 8 |
| uio_out | output | 8 |
| clk | clock | 1 |
| rst_n | input | 1 |
