# Tiny Tapeout N

**Source:** [https://github.com/Nama222/Tiny-Tapeout](https://github.com/Nama222/Tiny-Tapeout)

**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/3591](https://app.tinytapeout.com/projects/3591)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| RST_N | input | 1 |
| UI_IN | input | 8 |
| UO_OUT | output | 8 |

## First 10 Cycles

| Cycle | Phase | RST_N | UI_IN | UO_OUT |
|-------|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 | 0x0 |
| 1 | XOR Test (0^0) | 0x1 | 0x0 | 0x0 |
| 2 | XOR Test (1^0) | 0x1 | 0x1 | 0x0 |
| 3 | XOR Test (0^1) | 0x1 | 0x2 | 0x0 |
| 4 | MUX A (SEL=0, A=1, B=0) | 0x1 | 0x20 | 0x0 |
| 5 | MUX B (SEL=1, A=1, B=0) | 0x1 | 0xa0 | 0x0 |
| 6 | AND Test (CLK=1, IN4=1) | 0x1 | 0x10 | 0x0 |
| 7 | Idle | 0x1 | 0x0 | 0x0 |
