**Source:** [https://github.com/AAmirinejad/TinyTapeoutWorkshop](https://github.com/AAmirinejad/TinyTapeoutWorkshop)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| UI_IN | input | 8 |
| UO_OUT | output | 8 |

## First 10 Cycles

| Cycle | Phase | UI_IN | UO_OUT |
|-------|-------|-------|-------|
| 0 | 0 + 0 + 0 = 0, Cout=0 | 0x0 | 0x0 |
| 1 | 1 + 0 + 0 = 1, Cout=0 | 0x1 | 0x1 |
| 2 | 1 + 1 + 0 = 0, Cout=1 | 0x3 | 0x2 |
| 3 | 1 + 1 + 1 = 1, Cout=1 | 0x7 | 0x3 |
