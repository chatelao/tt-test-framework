## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| SWITCHES | input | 8 |
| SEGMENTS | output | 8 |
| CLK | clock | 1 |
| RST_N | input | 1 |

## First 10 Cycles

| Cycle | Phase | SWITCHES | SEGMENTS | RST_N |
|-------|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 | 0x0 |
| 1 | Secret Code | 0x42 | 0x0 | 0x1 |
| 2 | Other Code | 0xff | 0x0 | 0x1 |
