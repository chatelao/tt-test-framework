## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| UI_IN | input | 8 |
| CLK | input | 1 |
| RST_N | input | 1 |
| UO_OUT | output | 8 |

## First 10 Cycles

| Cycle | Phase | UI_IN | CLK | RST_N | UO_OUT |
|-------|-------|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 | 0x0 | 0x0 |
| 1 | Idle | 0x0 | 0x0 | 0x1 | 0x0 |
| 2 | Input Toggling 1 | 0x1 | 0x0 | 0x1 | 0x0 |
| 3 | Input Toggling 2 | 0x0 | 0x0 | 0x1 | 0x0 |
