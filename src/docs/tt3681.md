## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |
| clk | clock | 1 |

## First 10 Cycles

| Cycle | Phase | ui_in | uo_out |
|-------|-------|-------|-------|
| 0 | Sum 0 + 0 | 0x0 (input a=0, input b=0) | 0x0 (output 1=0, output 2=0) |
| 1 | Sum 1 + 0 | 0x1 (input a=1, input b=0) | 0x1 (output 1=1, output 2=0) |
| 2 | Sum 1 + 1 | 0x3 (input a=1, input b=1) | 0x2 (output 1=0, output 2=1) |
