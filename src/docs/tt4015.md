# UART interfaced 8ch PWM controller

**Source:** [https://github.com/ADDTDR/tt-pwm-controller](https://github.com/ADDTDR/tt-pwm-controller)

**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/4015](https://app.tinytapeout.com/projects/4015)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |
| uio_in | input | 8 |
| clk | clock | 1 |
| rst_n | input | 1 |

## First 10 Cycles

| Cycle | Phase | ui_in | uo_out | uio_in | rst_n |
|-------|-------|-------|-------|-------|-------|
| 0 | Reset | 0x0 | 0x0 | 0x0 | 0x0 |
| 1 | Idle | 0x0 | 0x0 | 0x0 | 0x1 |
