## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |

## First 10 Cycles

| Cycle | Phase | ui_in | uo_out |
|-------|-------|-------|-------|
| 0 | Group 1: (IN0 & IN1) | (IN2 & IN3) = UO0, UO1 | 0x0 (Input a=0, Input b=0, Input c=0, Input d=0, Input e=0, Input f=0, Input g=0, Input h=0) | 0x0 (Output y=0, Output z=0) |
| 1 | Group 2: (IN4 & IN5) | (IN6 & IN7) = UO6, UO7 | 0x0 (Input a=0, Input b=0, Input c=0, Input d=0, Input e=0, Input f=0, Input g=0, Input h=0) | 0x0 (Output y=0, Output z=0) |
| 2 | All group inputs high | 0x0 (Input a=0, Input b=0, Input c=0, Input d=0, Input e=0, Input f=0, Input g=0, Input h=0) | 0x0 (Output y=0, Output z=0) |
| 3 | All low | 0x0 (Input a=0, Input b=0, Input c=0, Input d=0, Input e=0, Input f=0, Input g=0, Input h=0) | 0x0 (Output y=0, Output z=0) |
