# RGB PWM

**Source:** [https://github.com/JohannesNekes/Tiny-Tapeout-RGB-PWM](https://github.com/JohannesNekes/Tiny-Tapeout-RGB-PWM)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ENA | input | 1 |
| RST_N | input | 1 |
| UI_IN | input | 8 |
| UO_OUT | output | 8 |
