## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ui_in | input | 8 |
| uo_out | output | 8 |
| uio_in | input | 8 |
| clk | clock | 1 |
| rst_n | input | 1 |

## First 10 Cycles

| Cycle | Phase | ui_in | uo_out | uio_in | rst_n |
|-------|-------|-------|-------|-------|-------|

| 1 | Idle | 0x0 | 0x0 | 0x0 | 0x1 |
