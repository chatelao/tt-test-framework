# SPI and I2C Register Bank

**Source:** [https://github.com/calonso88/dtu_tt_workshop](https://github.com/calonso88/dtu_tt_workshop)

**TinyTapeout Project Page:** [https://app.tinytapeout.com/projects/3675](https://app.tinytapeout.com/projects/3675)

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| ENA | input | 1 |
| RST_N | input | 1 |
| UI_IN | input | 8 |
| UO_OUT | output | 8 |
| UIO | inout | 8 |

## First 10 Cycles

| Cycle | Phase | ENA | RST_N | UI_IN | UO_OUT | UIO |
|-------|-------|-------|-------|-------|-------|-------|
| 0 | Reset | 0x1 | 0x0 | 0x0 | 0x0 | 0x16 |
| 1 | Idle SPI | 0x1 | 0x1 | 0x0 | 0x0 | 0x16 |
| 2 | SPI Select | 0x1 | 0x1 | 0x0 | 0x0 | 0x6 |
