# Verilog Multistage Oscillator with Enable and Counter

## Input/Output Definitions

| Signal | Type | Width |
|--------|------|-------|
| CLK | clock | 1 |
| RST_N | input | 1 |
| TAPS | input | 3 |
| OUT | output | 8 |
